# PyTorchServing
PyTorch Serving

## Requirements
 - [nvidia-docker](https://github.com/NVIDIA/nvidia-docker)
 - [git LFS](https://git-lfs.github.com/)

## Launching

Build & run docker:  

```bash
git clone
cd PyTorchServing
./docker.init.sh -i $GPU_INDEX
```  

Inside docker:  
```bash
./init.sh
CTRL+P+Q
```
